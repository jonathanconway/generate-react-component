# generator-react-component

# About

Generator for ReactJS Components

## Installation
```bash
# Make sure both are installed globally
npm install -g yo
npm install -g generator-react-component
```

## Generating new components
```bash
# After setup of course :)
# cd my-new-project
# Run the component generator
yo react-component HelloWorld
# ...or you can call it without params, to get prompted for the name...
# yo react-component
```
